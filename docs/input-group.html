# Input group

Group for inputs to control bar position (vertical / horizontal) and add margins

By default input-groups is vertical. For use horizontal add class isHorizontal to the parent.

Recommended to use for:

  • Checkbox
  • Input
  • Radio
  • Select
  • Switch

# Default group

Default example
