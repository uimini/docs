
## Yarn

```bash
yarn add uimini
```

## Build From Source

Optionally, you can build UImini from its source on Github. To do this, you'll need to have Node.js and npm installed. I use Gulp to build UImini.

```bash
git clone https://github.com/vedees/uimini.git
cd uimini
npm install
npm start
```
